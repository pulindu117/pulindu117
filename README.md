### Featured Projects

**[InsightForge](https://github.com/pulindu117/InsightForge)** — Autonomous AI Research Agent
> Built from raw Gemini API calls (no agent framework). Plans its own search queries, reads web sources via tool calling, self-corrects when tool-calling drifts, and writes a cited Markdown report. Includes a self-correction loop, input sanitisation against malformed LLM output, and an 11-test mocked pytest suite.

**[ResolveAI](https://github.com/pulindu117/ResolveAI)** — RAG-Based Customer Support Assistant
> Full-stack RAG system built from first principles (no LangChain). Covers document ingestion, chunking, embedding with BAAI/bge-small-en-v1.5, semantic retrieval via ChromaDB, a FastAPI backend, and a React + Tailwind frontend. Containerised with Docker Compose.

**[Fake News Detection](https://github.com/pulindu117/NLP_Group_04)** — NLP Group Project
> Project lead on a team of 3. Built and evaluated a Logistic Regression baseline and an LSTM model on the WELFake dataset (72,000+ articles) for binary fake news classification.
